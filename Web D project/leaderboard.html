HackVeta Leaderboard

The Leaderboard in HackVeta shows the ranking of participants based on their overall performance in the hackathon. Each submitted project is reviewed and scored by judges.

Judgement Criteria:

  • Innovation: How creative and original the idea is.
  • Technical Implementation: Code quality, logic, and successful working model.
  • UI / UX: Clarity, design, and ease of using the interface.
  • Problem Understanding: How effectively the solution solves the given challenge.
  • Presentation: Explanation and clarity of the final demo or report.

Based on these evaluation points, every team gets a total score. The leaderboard is updated according to this score, giving a clear view of which team performed best in the hackathon.
